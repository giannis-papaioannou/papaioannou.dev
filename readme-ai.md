## Overview

This project, [papaioannou.dev](https://papaioannou.dev), is a personal website and portfolio me. It serves as an online platform to showcase my skills, experience, and projects in software development.

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** TypeScript
- **Package Manager:** Npm
- **Container Runtime:** Docker

### Deployment

Deploy the project using **[docker compose](https://www.docker.com/):**

```sh
docker compose up -d --build
```
